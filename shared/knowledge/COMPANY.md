# 公司背景

---

## 业务定位

面向入境游旅行行业的海外社媒营销一站式解决方案。以「咨询服务 + AI Agent」组合切入，帮助不会做海外社媒的入境旅行企业从零建立海外获客能力。

- **Year 1**：通过咨询服务保障交付、积累行业数据资产
- **Year 2–3**：逐步提升 Agent 占比，最终实现 Agent 独立交付的垂直行业产品公司

**我们不是 MCN，不是代运营公司，不是 SaaS 工具商。**
我们是一家用 AI Agent 驱动的新一代垂直行业营销服务公司——用服务保障结果，用 Agent 构建效率壁垒。

---

## 目标市场

### 服务对象（按优先级）

- **核心客户（Year 1 主攻）**：中型专业入境旅行社（年营收 2,000 万+）、特色入境旅行社（定制游、深度游、小语种专门社，年营收 500 万+）
- **标杆客户（Year 1 重点突破）**：头部国企旅行社、地方文旅局、5A 景区集团、连锁酒店集团
- **未来扩展（Year 2+）**：China Travel 全品类供给侧——航司、免税零售、跨境消费等

### 目标客源国（帮客户触达的海外游客市场）

- **第一梯队**：北美（美国、加拿大）、日韩——入境游最大客源、内容数据最丰富、付费能力最强
- **第二梯队**：东南亚（新加坡、马来西亚、泰国）、俄罗斯——免签政策红利显著、增速最快
- **第三梯队**：欧洲（英国、德国、法国）、澳洲——高净值客群、客单价高但获客周期长

### 覆盖平台

TikTok、Instagram Reels、YouTube Shorts 为核心阵地，配合 Google/TripAdvisor 的 GEO（生成式引擎优化）搜索布局。

## 核心优势

### 与传统 MCN / 代运营公司相比

| 维度 | 传统 MCN | 我们 |
|------|---------|------|
| 生产方式 | 人力堆叠，多一个客户多招人 | AI Agent 驱动，多一个客户 Agent 更聪明 |
| 毛利结构 | 15–25%，越做越薄 | Year 1: 40%+，Year 3: 70%+，越做越厚 |
| 内容质量 | 依赖个人能力，人走质量就掉 | 基于数据归因持续优化，不依赖个人 |
| 行业深度 | 什么行业都接，不深入 | 只做入境游，积累行业专属数据和合规知识 |
| 客户粘性 | 低，客户随时可换 | 高，数据资产 + 工作流嵌入 + 合规依赖 |

### 与纯 AI 工具相比

| 维度 | 纯 AI 工具 | 我们 |
|------|-----------|------|
| 使用门槛 | 客户需要自己会用 | 客户只需确认，我们负责交付结果 |
| 行业适配 | 通用模型，不懂入境游 | 基于真实爆款数据训练的行业专属能力 |
| 合规保障 | 无 | 内置入境游合规知识库，零政治风险 |
| 服务保障 | 无 | 专属顾问陪跑，确保从 0 到 1 跑通 |

**一句话总结：** 传统 MCN 用人力做内容，我们用 Agent 做内容。纯 AI 工具给客户一个锤子让他自己钉钉子，我们直接帮他把房子盖好。

## 核心产品

### Layer 1 · Agent 基础包

**定位：** 市场渗透主力
**价格：** 9,800 元/月（月付，随时可退，含免费 0→1 启动咨询）

**解决的客户心理问题：** "靠不靠谱？我试得起吗？"——不到 1 万，试一个月，不满意随时停。

**客户需要提供：**
- 确认品牌调性和内容方向（启动咨询中完成）
- 提供基础素材（现有照片/视频，有多少给多少）
- 每月 1 次 30 分钟线上确认会

**交付物：**

1. **0→1 启动咨询（免费赠送）**
   前 2 周专属顾问陪跑：帮客户明确海外账号定位、品牌调性、目标客源国画像、内容矩阵规划。输出一份《海外社媒启动手册》，客户团队可留存使用。

2. **5 个核心 Agent 上线运行**
   - 选题 Agent：基于客源国热度 + 目的地数据自动生成每周选题
   - 内容制作 Agent：自动生成脚本、字幕、视频素材
   - 同行监控 Agent：基于客户状况监控同行动态
   - 客户洞察 Agent：基于客源国数据 + 行业趋势自动输出季度策略建议
   - 数据分析 Agent：爆款归因、竞对监控、投放优化
   客户无需操作，只需确认。

3. **覆盖 2 个平台、1 个语种地区陪跑**
   每周 1 次客户陪跑+指点。

4. **月度数据看板 + 增长复盘**
   每月自动生成数据报告：播放量、互动率、粉丝增长、内容效果归因。附带下月优化建议，客户可直接转发给老板/股东做汇报。

5. **每月固定算力**

**目标客户：** T2 中型入境社（年营收 2,000 万+）、T3 特色入境社（年营收 500 万+）

**转化目标：** 40–50% 在 Month 4–6 升级到 Layer 2

---

### Layer 2 · Agent 进阶包

**定位：** 利润引擎
**价格：** 28,000 元/月（季签 8.4 万 / 年签 28 万，送 2 个月）

**解决的客户心理问题：** "效果有了，怎么做更大？"——用 Layer 1 的真实数据驱动升级决策。

**客户需要提供：**
- 每季度 1 次 60 分钟策略评审会
- 配合 KOL 合作确认（Agent 推荐，客户审批）
- 提供更多目的地素材（可安排拍摄指导）

**交付物：**

1. **5 个 Agent 全矩阵部署**（与 Layer 1 同款，但全量运行形成完整闭环）
   - 选题 Agent / 内容制作 Agent / 同行监控 Agent / 客户洞察 Agent / 数据分析 Agent

2. **全平台、全区域人群陪跑**
   每周 2 次客户陪跑+指点，服务全平台全区域。

3. **海外 KOL 智能推荐 + 对接**
   Agent 自动推荐最匹配的 5–10 个海外旅行类 KOL/KOC，含历史合作 ROI 数据、粉丝画像匹配度评分。团队负责对接执行。

4. **季度增长策略报告**
   每季度输出深度报告：内容效果归因 → 客源国偏好变化 → 下季度选题预测 → 预算分配建议。客户向董事会/股东汇报的核心材料。

5. **GEO 搜索优化**
   针对 Google、TripAdvisor 等海外搜索引擎，优化客户目的地内容在 AI 搜索结果中的展示排名，覆盖海外游客"搜索—决策"环节的自然流量。

6. **每月固定算力**

**目标客户：** 从 Layer 1 升级的客户；直签的 T2 中大型入境社

**转化目标：** 30–40% 在 Year 2 升级到 Layer 3 或续签年度

**资产沉淀价值：** 每家客户贡献 20 条/月归因数据 + 3 个客源国偏好数据 + KOL 合作效果数据，构建客源国偏好模型和 KOL 资源池两大核心资产。

---

### Layer 3 · Agent 旗舰包

**定位：** 标杆客户
**价格：** 45,000–80,000 元/月（年签 + 绩效分成，每成交 1 个入境客户额外收 200 元）

**解决的客户心理问题：** "我要建立长期的海外获客竞争优势。"——全 Agent 矩阵 + 系统深度集成 + 专属团队。

**客户需要提供：**
- 开放 CRM / 订单管理系统 API 对接
- 每月 1 次线下策略评审 + 季度高管复盘
- 指定内部对接人全权配合

**交付物：**

1. **Agent 业务定制**
   基于业务需求定制 Agent + 客户专属模型微调（基于该客户历史内容数据训练，生成更贴合品牌调性的内容）。

2. **全语种、全平台陪跑**
   每周 2 次客户陪跑+指点，服务全平台全区域。

3. **CRM + 订单系统集成**
   视频 CTA 直接跳转预订页 → 归因每条内容带来多少询盘和订单 → 实现"内容—获客—成交"闭环数据。

4. **海外 KOL 矩阵运营**
   直接管理 10–20 人 KOL 矩阵：签约、内容共创、效果追踪、续约优化。相当于客户拥有一支海外"品牌大使团"。

5. **专属客户成功团队**
   1 名高级策略顾问 + 1 名内容审核主管 + 1 名数据分析师的专属三人组。每月线下到店 1 次。

6. **行业合规保障**
   所有内容发布前经合规知识库自动检测：地理敏感词、民族文化红线、平台审核规则、版权风险。出具合规检测报告，客户零政治风险。

**目标客户：** T1 头部国企旅行社、地方文旅局、5A 景区集团、连锁酒店集团

**转化目标：** Year 1 签 5–10 家，作为行业标杆案例

**资产沉淀价值：** 深度 CRM 集成形成工作流嵌入壁垒（客户转换成本极高）。行业标杆案例为 Year 2 向同类客户批量复制提供背书。

## 当前获客渠道

### 线上渠道
- **行业内容营销**：在环球旅讯、文旅头条、36氪等行业媒体发布入境游海外营销趋势文章，建立专业认知
- **微信生态**：公众号 + 视频号发布行业洞察和客户案例，朋友圈精准触达旅行社老板圈层
- **小红书 / 抖音**：发布入境游海外营销的教育内容，吸引 T3 小型特色入境社关注
- **免费竞争力报告（L0 产品）**：通过上述渠道引导客户填写 3 分钟问卷，自动生成定制化竞争力报告，是最核心的线上获客入口

### 线下渠道
- **行业展会**：ITB China、CITM（中国国际旅游交易会）、IT&CM 等入境游核心展会，现场收集高质量线索
- **行业协会合作**：与中国旅行社协会入境游分会、各省旅行社协会合作，触达会员单位
- **BD 直接拓展**：销售团队主动拜访目标客户，配合 L0 报告做敲门砖

### 转介绍
现有 L1/L2 客户向同行推荐，是 Year 1 后半程最高效的获客方式。
