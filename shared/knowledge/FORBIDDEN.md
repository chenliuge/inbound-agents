## 二、禁止的承诺（Agent 严禁在任何内容、方案、销售话术中使用）

| 禁止承诺 | 原因 | 合规替代说法 |
|---------|------|------------|
| "保证GEO排名前X名" | 算法不受控，承诺无法兑现 | "目标是6个月内进入主要AI推荐列表，多数客户在4-5个月实现" |
| "3个月询盘翻X倍" | 无法对所有客户保证，对不同基数客户差异巨大 | "这家年营收2500万的入境社，2个月内询盘翻了5倍"（具体案例说事实） |
| "稳定询盘XX条/月" | 入境游淡旺季波动大，做不到稳定数字 | "帮你建立稳定的英文内容资产，让海外客户在AI搜索里能找到你" |
| "包成交""未成交退款" | 把行业做低，旅游成交周期长，退款条件无法界定 | 可设合理对赌：GEO达标指标未完成退还部分服务费 |
| "保证月入XX万" | 不专业，入境游老板反感 | 用真实案例数据代替承诺 |
| "全网最低价" | 廉价感，目标客群不认同 | 不提价格排名，用TCO对比（比自建团队便宜）说明价值 |
| "XX天速成" | 对专业度的侮辱，入境游获客是3-6个月系统建设 | "前2个月看不到效果，第3个月才开始起量——这是必经过程" |
| "AI帮你轻松赚钱" | 情绪化过度承诺，老板不信 | 说清楚要投入什么、经历什么、才能拿到什么结果 |

## 三、效果对赌条款设计红线（如产品引入对赌模式，必须遵守）

- ❌ 无上限抽成：单客户效果分成必须设年度上限（建议不超过服务费×3倍）
- ❌ 无兜底条件：需约定客户最低配合义务（如每月提供X条素材、配合X次访谈）
- ❌ 对赌周期过长：效果对赌条款最长覆盖签约后6个月，不捆绑全年
- ❌ KPI定义模糊：对赌指标必须量化（如"英文ChatGPT搜索'China inbound tour'前3推荐"，而非"AI可见度提升"）
- ❌ 同时驻场超2家：驻场陪跑模式每人同时最多服务2家，否则交付质量无法保证

## 四、禁止的定位模糊（对外传播中必须避免）

| 禁止 | 原因 |
|------|------|
| 自称"AI营销服务商" | 泛化竞争，和今立智能/如此AI没有区分度 |
| 自称"全渠道数字营销" | 空洞，老板听了没感觉 |
| 案例不注明行业 | "帮某客户粉丝增长10倍"——如果不是入境游客户，一文不值 |
| 用竞品通用话术 | "赋能企业数字化转型""全链路AI营销闭环"等，见BRAND-VOICE禁用词 |
| 对外攻击竞品 | 点名竞品的缺点会引发争议，用"行业普遍问题"描述，而不点名 |

## 五、政策/合规红线

- 禁止发布未经核实的签证政策/免签政策信息（政策变动频繁，出错影响客户生意）
- 禁止引用无出处的行业数据（必须标注来源：携程、国家移民局、UNWTO等权威机构）
- 禁止以客户名义发布任何公开声明（如"XX旅行社老板说……"需获得书面授权）
- 禁止承诺具体价格（定价随业务调整，Agent输出的方案中不写固定报价，应写"请联系了解最新方案"）
- 禁止生成真实人物的伪造引语（即使是竞品老板，也不可编造其言论）

## 六、内容创作禁区（补充BRAND-VOICE）

- 不做通用营销内容：每篇内容必须有入境旅游行业的专属场景，不写"企业如何做AI营销"的通用教程
- 不炒时间敏感热点：AI搜索/GEO相关政策、平台算法变动，不在未经验证前发布"确定性"结论
- 不发竞品截图或排名：即使是表扬竞品，也不截图对比，避免被认为是广告或引发法律纠纷
- 不写"小白入门"类内容：目标读者是做了10年+的入境游老板，不是互联网小白；内容要在专业层面对话
