# EXTRACTION_FRAMEWORK：爆款拆解框架

> 图文 7 维 + 视频 8 维，作为**默认工具箱**。
> **框架服务分析，不是分析服从框架**：遇到特殊样本可自定义维度。

---

## 使用原则

1. **默认按本框架拆**：90% 情况下标准维度够用
2. **允许自定义**：遇到以下情况可跳出框架
   - 纯图片笔记（无文字正文）
   - 漫画/四格/表情包类
   - 极短视频（< 15 秒）或纯字幕卡片
   - 跨模态内容（多形式混合）
   - 框架维度套不进去反而牵强的
3. **自定义时的要求**（参见 RULES R4/R5/R5.1）：
   - 说明为什么标准框架不适用
   - 列出自定义维度清单
   - 每条拆解仍要给出可复用元素

## 一、图文爆款拆解（7 维）

适用：小红书图文笔记、公众号长文、微博图文等

### 维度 1：标题公式

识别爆款标题所属的公式类型，归类到以下之一（可多归属）：

| 类型 | 特征 | 示例 |
|---|---|---|
| 数字冲击型 | 带大数字、反差数字 | "3000 万老外免签入境" |
| 身份锁定型 | 明确指向某类人群 | "入境社老板必看" |
| 反差故事型 | 两件事对比 | "展会 20 万 vs GEO 5 万" |
| 打脸反转型 | "以为 XX 结果 YY" | "18 年入境游，第一次被 AI 打败" |
| 名单点名型 | "我搜了 XX" | "搜'中国旅游'，AI 推荐了 7 家" |
| 焦虑唤醒型 | "你还在 XX 吗" | "同行都在 TikTok 了，你还在 OTA" |
| 清单预告型 | "X 件事必做" | "3 件事让 ChatGPT 推荐你" |

**输出格式**：`标题公式：[数字冲击型 + 焦虑唤醒型]，核心词："3000 万 / 89%"`

### 维度 2：开头钩子（前 200 字）

分析正文开头 200 字的钩子类型：

| 类型 | 识别特征 |
|---|---|
| 饭局故事 | "上周跟 XX 吃饭" / "某老板跟我说" |
| 数据冲击 | 开篇抛大数字或反差数据 |
| 直接提问 | "你有没有想过..." |
| 打脸开场 | "我一开始以为...后来发现..." |
| 场景带入 | 描述一个具体场景 |
| 结论先行 | 第一句直接给结论 |

**输出格式**：`开头钩子：[数据冲击] 首句："3000 万老外免签进来，89% 没通过旅行社"`

### 维度 3：结构模板

识别正文的结构模板：

| 模板 | 特征 |
|---|---|
| 痛点 → 原因 → 解法 | 线性推进 |
| 3 段式 / 5 步法 | 明确编号的步骤 |
| 对比双线 | A 怎么做 vs B 怎么做 |
| 问题清单 | 抛 N 个问题逐一解答 |
| 案例倒叙 | 先讲结果再复盘过程 |
| 金句串联 | 多个金句连缀成文 |

**输出格式**：`结构模板：[5 步法]，5 个小标题：[01 做市场判断]/[02 建英文官网]/.../`

### 维度 4：视觉节奏

分析视觉呈现的节奏特征：

- **段落长度**：单段最长 X 字 / 平均 X 字
- **加粗频率**：每 N 字一次加粗 / 哪类内容加粗（数据/结论/引导）
- **列表使用**：是否用 ①②③ / • / `>` 引用块
- **emoji 密度**：全文 X 个 / 哪些位置用（标题/引导/小结）
- **图片布局**：封面 + N 张正文图 / 数据图表占比

**输出格式**：`视觉节奏：段落 ≤ 3 行；每 150 字一次加粗；用 ① 做清单；emoji 4 个（仅用于标题/CTA）`

### 维度 5：金句位置

识别爆款中可截图转发的金句：

- 数量：全文 N 个金句
- 位置：出现在第几段 / 相邻多少字
- 类型：
  - 观点金句："XX 本质上是 YY"
  - 判断金句："这件事的核心不是 X 而是 Y"
  - 数据金句："一个 GEO 投放的 ROI 是展会的 X 倍"
  - 场景金句（带画面感）

**输出格式**：`金句 4 个，平均每 500 字一个。金句类型：观点型 2 / 数据型 1 / 场景型 1。示例："GEO 不是 SEO 的升级，是 SEO 的谋杀"`

### 维度 6：CTA

分析文章结尾的引导设计：

- 明示 CTA：具体引导动作（关注 / 评论区关键词 / 后台回复 / 加微信）
- 暗示 CTA：软性引导（"想知道更多可以..."）
- 钩子诱饵：是否给"免费资料/报告/方案"
- 互动率提升设计：（选题投票 / 评论区提问 / 扣 666）

**输出格式**：`CTA：明示 - 后台回复"GEO"领取 SOP 文档；暗示 - 无；钩子诱饵 - 「入境游 GEO SOP」免费文档`

### 维度 7：话题标签（小红书专属）

如果是小红书笔记，分析：

- 标签数量（通常 3-5 个）
- 标签层级：大词（旅游） + 垂类（入境游） + 长尾（外国人来中国）
- 是否带热点标签

**输出格式**：`话题：#入境游 #老板必看 #营销方法论 #ChatGPT。层级组合：垂类 + 身份 + 品类 + 工具`

## 三、拆解输出格式

每条爆款的拆解必须输出为以下结构化 markdown：

```markdown
### [爆款 #N] [平台] [标题]

**URL**: ...
**数据**: 点赞 X / 收藏 X / 评论 X
**发布日期**: YYYY-MM-DD
**作者**: ...
**爆款级别**: 🟡 爆款（Layer 1 垂类）

**拆解**：

#### 维度 1：标题公式
...

#### 维度 2：开头钩子
...

...（所有维度）

**核心可借鉴元素**（3 条）：
1. ...
2. ...
3. ...

**不建议照抄**（若有）：
- ...
```
