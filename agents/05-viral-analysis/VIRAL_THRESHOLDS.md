## 评分函数（代码化）

```python
# 返回 "big_viral" / "viral" / "potential" / "normal"

THRESHOLDS = {
    "xhs": {"big": 10000, "viral": 2000, "potential": 500},       # 以点赞为主指标
    "wechat": {"big": 100000, "viral": 10000, "potential": 1000}, # 以阅读为主
}

VERTICAL_COEF = 0.3  # 垂类降权系数

def score_item(item: dict, platform: str, layer: int) -> str:
    """根据作品数据、平台、Layer 返回爆款级别。"""
    th = THRESHOLDS.get(platform, {})
    coef = VERTICAL_COEF if layer in (1, 2) else 1.0

    # 主指标：小红书点赞 / 公众号阅读
    main_value = {
        "xhs": item.get("stats", {}).get("like", 0),
        "wechat": item.get("read_count", 0),
    }.get(platform, 0)

    if main_value >= th.get("big", 0) * coef:
        return "big_viral"
    if main_value >= th.get("viral", 0) * coef:
        return "viral"
    if main_value >= th.get("potential", 0) * coef:
        return "potential"
    return "normal"
```

## 入选规则

**入选做拆解的标准**：
- Layer 1+2：达到**垂类"潜在爆款"**即可（降低门槛确保有样本）
- Layer 3+4：必须达到**基础"爆款"**（提高门槛保证"值得借鉴"）

**最终拆解名单**：
- 优先选 3 条 Layer 1+2 垂类潜在爆款以上
- 若不足 3 条，补 Layer 3+4 爆款至 3 条
- 最多 8 条（超过时按互动降序取前 8）
