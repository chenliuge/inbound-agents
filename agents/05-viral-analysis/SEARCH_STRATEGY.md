# SEARCH_STRATEGY：四层递进式爆款搜索

> 入境游垂类样本稀少，单层搜索命中率低。采用四层递进策略，逐步从精准垂类扩展到结构通用层。

---

## 整体逻辑

```
选题: [输入，如"入境游 GEO 4 步法"]
   ↓
Layer 1: 精准垂类（入境游本身）
   ↓ 命中 < 5 条有效爆款 → 触发 Layer 2
Layer 2: 相邻垂类（出入境/旅游 B 端）
   ↓ 命中累计 < 10 条 → 触发 Layer 3
Layer 3: 相关行业（B 端营销/跨境/AI 工具）
   ↓ 命中累计 < 15 条 → 触发 Layer 4
Layer 4: 结构通用层（任意行业，按爆款结构类型）
   ↓
汇总所有 Layer 样本 → 按 VIRAL_THRESHOLDS 评分 → 输出
```

---

## Layer 1：精准垂类

**目的**：找到**入境游本身**的爆款。若找到，直接借鉴"内核 + 结构"。

**关键词生成规则**（基于选题）：
从选题中提取：
- 核心业务词：入境游、外国游客、inbound tourism、China travel
- 场景词：签证、免签、海外社媒、客源国、海外营销
- 方法词：SOP、攻略、ROI、询盘、获客

**关键词示例**（选题"入境游 GEO 4 步法"）：
- `入境游 GEO`
- `外国游客 AI 搜索 中国`
- `中国旅行社 ChatGPT 推荐`
- `入境游 营销 AI`
- `inbound China SEO`

**搜索接口**（当前只做小红书 + 公众号图文）：
- 小红书：`xhs_search_note(kw)` + `xhs_note_detail(note_id)` 拿完整正文
- 公众号：`wechat_kw_search(kw)` + `wechat_hot_typical_search(kw)`
- 账号池：`xhs_user_post(uid)` + `wechat_post_history(name)`

**样本阈值**：至少找到 3 条 **垂类降权后的爆款**（见 VIRAL_THRESHOLDS），才算 Layer 1 有效样本。

---

## Layer 2：相邻垂类（扩展触发条件：Layer 1 < 5 条）

**目的**：借"邻居"的经验，通常结构相通、内核接近。

**邻居域**：
- **出境游**：中国人出国旅行相关（叙事角度相似）
- **留学 B 端**：海外获客模式相似
- **海外营销工具**：操作方法相似
- **旅行社经营**：客户群体重叠

**关键词生成**：
从 Layer 1 的关键词派生：
- 将"入境游" → 替换为"出境游 / 留学 / 海外营销 / 旅行社"
- 保留方法词（SOP/GEO/ROI 等）

**关键词示例**：
- `出境游 GEO`
- `留学机构 ChatGPT 招生`
- `海外营销 AI 工具`
- `旅行社 海外获客`

**搜索接口**：同 Layer 1，但范围放宽。

---

## Layer 3：相关行业（扩展触发条件：Layer 1+2 累计 < 10 条）

**目的**：到此已经脱离"旅游"语境，但保留"B 端服务/海外市场/AI 赋能"这类**商业模式相似**的内容。

**相关域**：
- **B 端 SaaS 营销**：面向企业卖软件的内容
- **跨境电商**：面向海外消费者卖货的内容
- **外贸**：B2B 贸易获客
- **海外 DTC 品牌**：消费品出海

**关键词生成**：
- 完全去掉"旅游"语境
- 保留"海外 / 出海 / AI / 获客 / 转化 / ROI / 老外"
- 替换垂类词：旅行社 → 外贸公司 / 品牌

**关键词示例**：
- `跨境电商 GEO`
- `外贸 ChatGPT 引流`
- `SaaS 出海 获客`
- `DTC 品牌 海外 AI`

---

## Layer 4：结构通用层（扩展触发条件：Layer 1+2+3 累计 < 15 条）

**目的**：完全不限行业，按**爆款结构类型**找。不借鉴内容，只借鉴骨架。

**结构类型**（从选题中识别）：
根据选题标题/内容，识别其结构模板，然后搜该结构的任意爆款：

| 结构类型 | 关键词模式 | 示例搜索词 |
|---|---|---|
| X 步法/SOP | "X 步"、"SOP"、"方法论" | `3 步搞定`、`5 步 SOP`、`方法论 爆款` |
| 反差对比 | "vs"、"差距"、"差在哪" | `20 万 vs 5 万`、`对比 爆款` |
| 身份锁定 | "XX 必看"、"老板"、"自救" | `老板必看 爆款`、`创始人 自救` |
| 数据冲击 | 大数字、反差数字 | `3000 万`、`89%`、`一条 xx` |
| 故事反转 | "第一次"、"结果没想到" | `第一次`、`以为 结果`、`打脸 故事` |
| 清单体 | "X 件事"、"X 条"、"清单" | `3 件事 必做`、`10 条 清单` |
| 名单点名 | "搜了 XX"、"推荐了" | `AI 推荐 谁`、`榜单 点名` |

**关键词生成**：
- 只包含结构词，不包含行业词
- 在小红书/抖音/公众号搜

**关键词示例**（选题是"4 步法"）：
- `4 步 SOP 爆款`
- `4 步搞定 方法论`
- `4 步 建议收藏`

---

## 扩展规则细节

### 关键词衍生（抖音 search_sug）
每个 Layer 的每个关键词，调用 `douyin_search_sug(kw)` 拿衍生词，扩充到该 Layer 的关键词池。

### 并发控制
- 每 Layer 最多生成 **10 个关键词**
- 每关键词每平台最多拿 **20 条结果**
- 单次分析总请求数上限 **100 次**（防 QUOTA 爆炸）

### 去重
按 `note_id` / `aweme_id` / URL 去重。同一条爆款在不同关键词下被重复命中只计一次。

### 提前终止
如果 Layer 1 已经找到 ≥ 10 条有效样本 → 无需触发 Layer 2。
但 Layer 1+2 的有效样本 < 10 条时仍会继续扩展。

---

## 置信度判定（汇总所有 Layer 后）

| 条件 | 置信度 | 处理 |
|---|---|---|
| Layer 1+2 各含 ≥ 3 条爆款 | 🟢 高 | 正常输出 |
| Layer 1+2 合计 ≥ 3 条爆款 | 🟡 中 | 正常输出，标注"垂类样本偏少" |
| Layer 1+2 < 3 条，Layer 3+4 ≥ 3 条 | 🟡 中 | 以 Layer 3+4 为结构参考，提示"纯结构借鉴" |
| Layer 1+2+3+4 合计 < 3 条 | 🔴 低 | **触发人工补充**，不输出完整分析 |

**低置信度触发人工补充时的消息模板**：
```
⚠️ 该选题找不到 3 条以上爆款。

已搜索层级：
  Layer 1 精准垂类：[N] 条
  Layer 2 相邻垂类：[N] 条
  Layer 3 相关行业：[N] 条
  Layer 4 结构通用：[N] 条

已搜索关键词：[...]

请为我提供：
• 你看到的爆款链接（任意平台）
• 或告诉我类似选题（让我换个关键词重搜）
```
