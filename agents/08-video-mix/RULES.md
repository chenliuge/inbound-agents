# 规则

## 必须遵守

1. **每条视频必须包含 5 大板块，一个不能少**：节奏板 / 分镜表 / BGM 方案 / 字幕规范 / 素材清单
2. **平台合规优先**：先读 `shared/knowledge/PLATFORM-RULES.md`，严格按目标平台的时长、分辨率、字幕安全区、版权音乐规则来产出规格
3. **分镜时长总和 = 视频总时长 ± 2 秒**，不能出现总和对不上的情况
4. **每个转场必须注明理由**：为什么这里用硬切、为什么那里用叠化，不能只写"硬切"
5. **BGM 用情绪标签 + 方向描述**，**严禁**直接指定有版权的热门曲目（如"周杰伦某某歌"），避免商业账号被静音
6. **字幕双语规则**（面向海外平台）：主字幕用目标客源国语言（如英文），辅助字幕用中文（给团队自审用），标注清楚哪条是哪条
7. **素材清单必须明确标注每段素材的获取方式**：原始（客户提供）/ 新拍（需排期）/ AI 生成（给出 prompt）/ 外购（素材站名 + 关键词）
8. **关键信息（景点名、价格、日期、CTA）必须在字幕中高亮**，高亮方式与平台字幕样式库保持一致

## 严禁

- 指定特定的商业版权音乐（只给情绪方向，让剪辑用平台商业音乐库匹配）
- 产出无法执行的艺术化描述（"有电影感的镜头"这种空话，必须翻译成具体参数）
- 忽略目标平台的时长上限（如 TikTok Shorts 不超过 60 秒建议，Reels 不超过 90 秒）
- 漏写字幕安全区（顶部/底部 UI 遮挡区域），导致关键字被 UI 盖住
- 生成含有政治敏感、文化冒犯、版权风险的内容指引
- 建议搬运未授权的第三方视频素材
