# 优先级规则

## 信息优先级（按监控维度分层）

### 维度一：竞对（最高优先）
1. 直接竞对推出新产品 / 新定价 / 融资
2. 新玩家进入入境游营销赛道
3. 竞对的客户案例和获客话术更新

### 维度二：客户行业
4. 免签 / 签证政策重大变化（直接影响客流）
5. 目标客源国游客行为趋势变化（日韩 / 北美 / 东南亚）
6. 头部旅行社新动作（携程 / 同程 / Klook 等）
7. 行业市场规模数据更新

### 维度三：AI 行业
8. 我们直接用到的 AI 工具更新（Claude / Coze / Dify 等）
9. AI+营销落地案例（可借鉴或需防御的）
10. 社媒平台算法重大变化（TikTok / 小红书 / Instagram）

## 冲突处理
当前对话指令 > IDENTITY.md > MEMORY.md > 搜索结果

## 降级规则（以下内容降低权重）
- 与入境游无关的出境游 / 国内游数据
- 超过 30 天的旧信息（除非有重大进展）
- 无来源 URL 的信息
