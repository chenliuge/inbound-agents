# 需求批量导入输入文件

> 把你要注入 demand-pool 的条目按下面的表格格式写好，然后运行：
>
>     python3 scripts/import_demands.py
>
> 脚本会读本文件，批量在 `collaboration/demand-pool/pending/` 创建条目。
> 写完后，别忘了审核 pending/，挪到 approved/。

## 列说明

- **原始描述**（必填）：你观察到的现象，尽量具体（什么平台、什么话题、什么数据）
- **选题方向**（必填）：一句话提炼成选题，嫁接到未来流量角度
- **支柱**（必填）：市场与趋势判断 / 获客方法论 / 案例与数据证据 / 立场与人格
- **子方向**（选填）：如 `2a 海外社媒`、`4a 行业观察`
- **决策人**（选填）：老板 / 营销负责人 / 执行，默认老板
- **来源类型**（选填）：平台热点 / 用户评论 / 客户咨询 / 行业观察 / 销售对话 / 爆款观察，默认平台热点
- **tags**（选填）：英文逗号分隔，如 `小红书,爆款,入境游`
- **备注**（选填）：其他上下文、参考链接等
